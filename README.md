# ATM simulator (C++)
ATM simulator developed in C++ as a learning project.

## Objectives
- Practice Object-Oriented Programing (OOP).
- Improve programing logic.
- Simulate real ATM operations.

## Features (planned)
- User login.
- Balance inquiry.
- Withdraw and deposit money.
- Transfers.
- Service payments.

## Project status
In progress (Module 1)
